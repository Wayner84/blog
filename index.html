Posting workflow (simple on purpose)
  1. Copy posts/example-post-10.02.26.html
  2. Rename it to posts/TITLE-DD.MM.YY.html
  3. Edit the markdown inside <script type="text/markdown" id="post-markdown">
  4. Drop photos into assets/images/ and reference them in markdown
  5. Add the new entry to posts/manifest.json (top = newest), then push

No build step. No database. Just commit and it’s live.
